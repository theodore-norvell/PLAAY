bind

  • bind<B>(f: function): List<B>
    • Defined in collections.ts:182

    The monadic bind or flat map.

    This function applies maps the given function, which must map each element to a list and then flattens the result.

    For example if list(a,b,c).map(f) gives list([a0,a1],[], [c0,c1,c3]), then list(a,b,c).bind(f) gives list(a0,a1,c0,c1,c3).

    Choose an action based on whether the list is empty.

    l.choose(f, g) is the same as g() if the list is empty. Othewise, it is the same as f(h, r) where h is the first item and r is the rest of the list.
