tryModify

  • tryModify(newChildren: Array<PNode>, start: number, end: number): Option<PNode>
    • Defined in pnode.ts:117

    Possibly return a copy of the node in which the children are replaced. The result will have children [c[0], c[1], c[start-1]] ++ newChildren ++ [c[end], c[end+1], ...] where c is this.children(). I.e. the segment c[ start,.. end] is replaced by newChildren. The method succeeds iff the node required to be constructed would be valid. Node that start and end can be number value including negative. Negative numbers k are treated as length + k, where length is the number of children.

    Parameters

    • newChildren: Array<PNode>

      An array of children to be added

    • start: number

      The first child to omit. Default 0.

    • end: number

      The first child after start to not omit. Default this.children().length.

    Returns Option<PNode>
