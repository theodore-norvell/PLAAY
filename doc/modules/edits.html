alt

  • alt<A>(first: Edit<A>, ...rest: Edit<A>[]): Edit<A>
    • Defined in edits.ts:103

    A biased choice between or one, two, or more edits. Given var z = alt(x,y) ;

    • z.applyEdit(a) is the same as x.applyEdit(a) if that is successful.
    • z.applyEdit(a) is the same as y.applyEdit(a) if x.applyEdit(a) is not successful. Given var z = alt( v, w, x, y) z tries each edit in order until one is successfull or all have failed.

compose

  • compose<A>(first: Edit<A>, second: Edit<A>): Edit<A>
    • Defined in edits.ts:74

    The composition of two edits. Does one edit and then the other. Given var z = compose(x,y) ;, z.applyEdit(a) applies x to a and then applies y to the result of that; but z.applyEdit(a) fails if either either application fails.

id

  • id<A>(): Edit<A>
    • Defined in edits.ts:123

    An edit that does nothing. This is useful in combination with alt. For example given var z = alt( x, id() );, z.applyEdit( a ) is the same as x.applyEdit(a) if that succeeds. It is Some( a ) if x.applyEdit(a) fails.
