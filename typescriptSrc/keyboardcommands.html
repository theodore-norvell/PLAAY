Keyboard commands:

The following keyboard commands can be used creating code:

  • ctrl+shift+v will create a variable declaration.
  • shift+; (:) will create an assignment expression.
  • shift+' (") will create a string literal expression.
  • shift+/ (?) will create an alternation (if) expression.
  • shift+2 (@) will create a while-loop expression.
  • \ will create a lambda expression.
  • shift+- (_) will create an application expression
  • Typing any number will create a numeric literal expression starting with that number. Works with the numpad as well.
  • Typing any alphabet character will create a variable expression starting with that character.
  • Typing +, -, *, or / will create the appropriate call expression. Also works with shift+= for + and shift+8 for *.

You can use the arrow keys and tab key to navigate your code.

The space bar will expand the current selection outward.

You can use ctrl+a to select everything.

You can also use ctrl+c, ctrl+v, and ctrl+x to copy, paste, and cut selections from/to your code.

You can use ctrl+b to swap the current selection with the top of the trash stack.

You can use ctrl+1, ctrl+2, and so on to paste from anywhere in the trash stack. 1 will paste the second item down, 2 the third item, and so on. 0 acts the same as ctrl+v.

The cmd key can be used in place of ctrl for mac users.
