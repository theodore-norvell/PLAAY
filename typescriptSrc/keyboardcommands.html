Keyboard commands:

The following keyboard commands can be used for node creation:

  • ctrl+shift+v will create a variable declaration node.
  • shift+; (:) will create a variable assignment node.
  • shift+' (") will create a string literal node.
  • shift+/ (?) will create an if node.
  • shift+2 (@) will create a while node.
  • Typing any number will create a numeric literal node starting with that number. Works with the numpad as well.
  • Typing any alphabet character will create a variable node starting with that character.
  • Typing +, -, *, or / will create the appropriate world call node. Also works with shift+= for + and shift+8 for *.

You can use the arrow keys to navigate your code.

You can use ctrl+a to select everything.

You can use ctrl+b to swap the current selection with the top of the trash stack.

You can also use ctrl+c, ctrl+v, and ctrl+x to copy, paste, and cut selections from your code.

You can use ctrl+1, ctrl+2, and so on to paste from anywhere in the trash stack. 1 will paste the second item down, 2 the third item, and so on. 0 acts the same as ctrl+v.

The cmd key can be used in place of ctrl for mac users.
