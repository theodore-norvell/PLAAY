Keyboard commands:

On Macs and ipads, the cmd key can be used in place of the ctrl key.

The following keyboard commands can be used for creating declarations:

  • ; will create a location declaration. It will also convert a constant declaration to a location declaration.
  • , will create a constant declaration. It will also convert a location declaration to a constant declaration.

The following keyboard commands can be used for creating expressions:

  • : will insert an assignment expression.
  • " will create a string literal expression.
  • ? will insert an alternation (if) expression.
  • @ will insert a while-loop expression.
  • \ will insert a lambda expression.
  • _ will insert an application expression
  • 0-9 will insert a numeric literal expression starting with that number. Works with the numpad as well.
  • a-z and a-z will insert a variable expression starting with that character.
  • +, -, *, /, <, >, % and = will create the appropriate call expression. Also works with shift+= for + and shift+8 for *.
  • ` will insert a call expression.
  • $ will insert an object literal expression.
  • ( will insert a tuple expression.
  • [ will insert an accessor expression.
  • ` will insert a call expression.

The following keyboard commands can be used for creating types:

  • " will insert the string type.
  • ? will insert the boolean type.
  • \ will insert the function type.
  • 0 will insert the natural number type.
  • 1 will insert the integer type.
  • 2-9 will insert the number type.
  • ; will insert a location type.
  • : will insert a field type.
  • ( will insert a tuple type.
  • & will insert a meet type.
  • | will insert a join type.

Arrow keys and the tab key to navigate your code.

Arrow keys while shift key is depressed to expand or contract the selection.

Space bar to expand the current selection outward.

Enter to edit the text associated with a node.

ctrl+a to select everything.

ctrl+c to add the current selection to trash.

ctrl+x to cut the current selection to trash.

ctrl+v to paste the top (most recently added) item from the trash.

ctrl+b to swap the current selection with the top of the trash.

crtl+0, ctrl+1, ..., ctrl+9 to paste an item from the trash. 0 will paste the top trash item. 1 will paste the second trash item, 2 the third item, and so on.
