Keyboard commands:

On Macs and ipads, the cmd key can be used in place of the ctrl key.

Creating expressions with the keyboard

Infix/postfix example: x enter , tab 2 enter * y enter ` f enter space + 1 tab

Prefix example: , x tab tab + * 2 tab ` f tab y tab tab tab 1 tab

  • 0-9 will create a numeric literal expression starting with that digit. Works with the numpad as well.
  • a-z and A-Z will create a variable expression starting with that character.
  • " will create a string literal.
  • ` will create a call expression.
  • . will create a selection expression.
  • For each of the above use tab or enter to finish.
  • +, -, *, /, <, >, %, &, |, and = will create the appropriate call expression.
  • ? will create an alternation (if) expression.
  • @ will create a while-loop expression.
  • \ will create a lambda expression.
  • _ will create an application expression
  • ( will create a tuple expression.
  • $ will create an object literal expression.
  • [ will create an index expression.
  • : will create an assignment expression.
  • ; will create a loc expression.

Creating declarations with the keyboard

Variable declaration example: a b c enter , or , a b c tab

Location variable declaration example: a b c enter , ; or , a b c enter ;

For function parameters, the comma (,) is omitted.

  • , will create a variable declaration.
  • ; will turn plain variable declarations into location variable declarations (or the opposite) when the declaration is selected.

Moving around with the keyboard

  • ← → ↓ ↑ to change the selection.
  • shift+ ← → ↓ ↑ to expand or contract the selection.
  • space to expand the current selection outward.
  • enter to edit the text associated with the selection.
  • tab to move the selection to the next dropzone or place-holder.
  • shift+tab to move the selection to the previous insertion point.
  • ctrl+a to select everything.

Other editing with the keyboard

  • ctrl+c to add the current selection to trash.
  • ctrl+x to cut the current selection to trash.
  • ctrl+v to paste the top (most recently added) item from the trash.
  • ctrl+b to swap the current selection with the top of the trash.
  • crtl+0, ctrl+1, ..., ctrl+9 to paste an item from the trash. 0 will paste the top trash item. 1 will paste the second trash item, 2 the third item, and so on.
